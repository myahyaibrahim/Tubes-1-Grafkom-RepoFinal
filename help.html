Panduan Penggunaan Web Implementasi Sederhana WebGL

Deskripsi website

Website hasil implementasi WebGL sederhana ini merupakan sebuah webiste yang dapat membuat model poligon untuk kebutuhan denah arsitektur

Model-model yang dapat digambar berupa

  • Garis
  • Persegi (Segiempat sama sisi)
  • Persegi panjang (tidak harus sama sisi)
  • Poligon

Model-model yang telah dibuat dapat disimpan dan di-load oleh website

Untuk menyimpan model yang telah dibuat, dapat dnegan menekan tombol save, tunggu sebentar, dan tulisan download akan muncul di bawah

Untuk menjalankan file model hasil penyimppanan dapat dilakukan dengan menempatkan file di folder yagn sama dengan kode program website

Fitur-Fitur Website

Menggambar model

Untuk menggambar model dapat dengan cara berikut

  1. Pilih jenis shape pada pilihan menu dropdown (Line, Squre, Rectangle, Polygon)
  2. Mulai menggambar dengan mengklik untuk menandai simpul model menggunakan mouse pada area canvas
  3. Untuk model polygon, untuk menggambar simpul terkahir dapat dilakukan dengan mengklik mouse dan menekan tombol shift pada keyboard secara bersamaan

Menggeser objek

Untuk menggeser objek dapat dilakukan dengan cara berikut

  • Pilih model yang akan digeser posisinya (1..N)
  • Urutan model gambar sesuai dengan jenis/shape model
  • Masukan nilai koordinat X,Y sebagai koordinat baru untuk model
  • Untuk model garis(line), pengguna dapat menggeser posisi koordinat X,Y dari masing-masing titik
  • Untuk model persegi dan persegi panjang, titik acuan dari X,Y adalah titik tengah model
  • Untuk model polygon, titik acuan yang digunakan adalah titik pertama yang tercatat ketiak menggambar model tersebut
  • Klik tombol Translation untuk melihat perubahan

Mengubah ukuran model

Untuk mengubah ukuran objek dapat dilakukan dengan cara berikut

  • Pilih model yang akan diubah ukurannya
  • Pada bagian Scaling, masukan nilai pengali X,Y yang akan diguanakan untuk mengubah ukuran objek model
  • Secara default, nilai pengali masing-masing untuk sumbu X dan Y adalah 1
  • Klik tombol Scale untuk melihat perubahan

Mengubah warna objek model

Untuk mengubah warna objek dapat dilakukan dengan cara berikut

  • Pilih model yang akan diubah warnanya
  • Pada bagian Color, masukan nilai warna sesuai dengan urutan RGB-A
  • Klik tombol Change Color untuk melihat perubahan
